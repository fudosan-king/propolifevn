Change or prevent updating Last Modified date for each post when editing from admin panel

== Description ==

<p>Using this plugin you can easily change or prevent changing last modified date for each post. Changing last modified date can come in handy when you would like to control "order by last modified" for posts (or custom post type).</p>

<p>Plugin will attach Modified On option under the Published On in Publish meta box on edit post screen in admin panel. Edit link next to the date can be used to change the modifeid date and "Don't update for this post" checkbox will stop modified date from being updated each time when changes are saved.</p>

== Installation ==
1. Upload "update-modification-date.zip"  (unzipped) to the "/wp-content/plugins/" directory OR zip package directly in Plugins > Add New > Upload menu in admin of wordpress
2. Activate the plugin through the "Plugins" menu in WordPress.
